What is Linty?

Linty is linting-as-a-service.

Users sign in with their GitHub account, and select repositories they'd like Linty to review. Linty then watches all commits to that repository and checks the code style to see if it adhere's to the project's own rules, or Linty's defaults. Pull requests on watched repositories can receive status checks indicating the pass/fail status of the latest commit. Repository owners can make Linty required, forcing all new code to adhere to a style code, resulting in a consistent style throughout the codebase.

Linty is maintained by ZeroCater and was created to assist with keeping our own projects consistent and clean. Because Linty is open source, anyone with experience with Python/Django is welcome to contribute. You can view the project's Trello board here.
