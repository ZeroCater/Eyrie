What is Eyrie?

Eyrie is our attempt to build what we thought the GitHub wiki should be, using GitHub pages as an example. Users can create documentation for their repo by creating an orphan branch in the repository and putting documentation there. When a user adds their repo to Eyrie, they can select the branch they'd like Eyrie to follow. Because Eyrie uses a branch in the repository itself, users can use pull requests and other git workflows. The repository can have directories, rather than just a flat list of documents. Eyrie also allows users to search across multiple repositories.

Eyrie is maintained by ZeroCater and was created in response to our growing need to maintain and browse documentation, while keeping our standard GitHub workflows in place. We wanted our documentation to be treated like code. Because Eyrie is open source, anyone with experience with Python/Django is welcome to contribute.

Eyrie was created as a fork of Linty, another open-source project of ours, and can be found on GitHub here.
